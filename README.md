# Laundrymart
Laundry Wallah - HTML Project
About This Project
Laundry Wallah is a fully structured HTML webpage built as part of an HTML fundamentals assignment. It demonstrates the use of semantic HTML5 elements to build a real-world business webpage.
Project Structure
Laundry/
└── index.html    → Main webpage
└── README.md     → Project documentation
Sections Included
Header - Welcome heading and navigation links
About - Description of the business
Image - Laundry facility photo with caption
Services - List of all services offered
Price List - Table with services, prices and delivery time
Book Now Form - Booking form with name, email, phone, service, date and notes
Footer - Contact information and copyright
How to View the Webpage
Option 1 - Open Locally
Download or clone this repository

Double click index.html

HTML Concepts Used
Semantic HTML5 tags: <header>, <main>, <footer>, <section>, <nav>, <figure>, <figcaption>
Table with <thead>, <tbody>, <tfoot>
Form with required attributes on all input fields
Ordered and unordered lists
Images with alt text
Navigation with anchor links
